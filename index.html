What is HopiBot

Nowadays, modern consumers want fast, easy access to information which HopiBot is an answer for that:

  • Provide answers to frequently asked questions quickly and efficiently
  • Allow patient's relatives to consult with the doctor directly
  • Track patients' care to reduce re-admissions
  • Protect user from the pandemic situation

01

Handle First-Level Question

The system will show a symptom report of the patience sequentially by using the patient code by using the ID and PWD given by the Hospital or register a new one.

02

Track Patients

The system will show a symptom report of the patience sequentially by using the patient code. Patient also can directly chat with a doctor in the browser by using the ID and PWD given by the Hospital.

03

Avoid physical contact

Hopibot is one of the healthcare providers to give advice to a patient. This also protect user from the pandemic situation
